# SlothfulDevLog Brand Guide

## ブランド名

**SlothfulDevLog**（スロースフル・デブログ）

## コンセプト

「Slothful = 怠惰」は欠点ではなく、戦略。
力を抜いて、自分のペースで、リアルなプロセスを記録する。
完璧に仕上げてから出すのではなく、途中経過こそがコンテンツ。

## トーン & ボイス

- 肩の力が抜けている
- 正直で飾らない
- 知的好奇心が滲み出る
- 「教える」よりも「一緒に考える」
- 専門用語を使いすぎない

## デザイン方針

Aesop（イソップ）のミニマルデザインを参考に。

### カラーパレット（仮）
- **背景**: オフホワイト / クリーム (#FAFAF5)
- **テキスト**: チャコール (#2D2D2D)
- **アクセント**: テラコッタ (#C87941)
- **サブ**: セージグリーン (#8B9A6B)
- **ボーダー**: ライトグレー (#E5E5E0)

### タイポグラフィ
- **見出し**: セリフ体（格調・落ち着き）
- **本文**: サンセリフ体（読みやすさ）
- **余白**: たっぷりと

### デザイン原則
1. Less is more — 要素を減らす
2. Typography first — 文字で語る
3. Breathing room — 余白で呼吸する
4. Authentic imagery — 自分で撮った写真、自分の記録
